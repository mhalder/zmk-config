## Num Layer (ENT hold)

Tap-dance on number keys:

- **Tap** = number (1, 2, 3...)
- **Double-tap** = symbol (!, @, #...)

For i3 workspaces: tap sticky SHIFT+GUI (right thumb), then tap number.

```
┌─────┬─────┬─────┬─────┬─────┬─────┐   ┌─────┬─────┬─────┬─────┬─────┬─────┐
│     │     │ 7/& │ 8/* │ 9/( │  [  │   │     │     │     │     │     │     │
├─────┼─────┼─────┼─────┼─────┼─────┤   ├─────┼─────┼─────┼─────┼─────┼─────┤
│     │     │ 4/$ │ 5/% │ 6/^ │  ]  │   │     │CTRL │ GUI │ ALT │     │     │
├─────┼─────┼─────┼─────┼─────┼─────┤   ├─────┼─────┼─────┼─────┼─────┼─────┤
│     │     │ 1/! │ 2/@ │ 3/# │  {  │   │     │     │     │     │     │     │
└─────┴─────┴─────┼─────┼─────┼─────┤   ├─────┼─────┼─────┼─────┴─────┴─────┘
```
